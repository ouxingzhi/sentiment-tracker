# 📈 财经舆情分析系统

实时监控财经新闻和社交媒体，分析市场情绪，预警异常动向。

## 🏗️ 系统架构

```
┌──────────────────────────────────────────────────────────────┐
│                     数据采集层 (Collectors)                    │
├────────────┬────────────┬────────────┬────────────┬──────────┤
│  财经新闻   │   微博     │   雪球     │   股吧     │  Twitter │
│  RSS/API   │   API      │   爬虫     │   爬虫     │   API   │
└────────────┴────────────┴────────────┴────────────┴──────────┘
                           ↓
┌──────────────────────────────────────────────────────────────┐
│                     消息队列 (Redis/RabbitMQ)                  │
└──────────────────────────────────────────────────────────────┘
                           ↓
┌──────────────────────────────────────────────────────────────┐
│                     处理分析层 (Workers)                       │
├────────────┬────────────┬────────────┬──────────────────────┤
│  文本清洗   │  情感分析   │  实体识别  │  趋势检测/预警        │
└────────────┴────────────┴────────────┴──────────────────────┘
                           ↓
┌──────────────────────────────────────────────────────────────┐
│                     存储层 (PostgreSQL)                       │
├────────────┬────────────┬────────────┬──────────────────────┤
│  原始文章   │  分析结果   │  股票实体  │  告警记录             │
└────────────┴────────────┴────────────┴──────────────────────┘
                           ↓
┌──────────────────────────────────────────────────────────────┐
│                     输出层                                     │
├────────────┬────────────┬────────────────────────────────────┤
│  FastAPI   │  Dashboard │  Telegram推送                      │
└────────────┴────────────┴────────────────────────────────────┘
```

## 🚀 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入API密钥

# 初始化数据库
python scripts/init_db.py

# 启动服务
docker-compose up -d
```

## 📦 核心模块

| 模块 | 功能 | 文件 |
|------|------|------|
| 数据采集 | 多源财经资讯抓取 | `src/collectors/` |
| 情感分析 | NLP情感判断 | `src/analyzers/` |
| 实体识别 | 股票/公司/人物识别 | `src/analyzers/` |
| API服务 | RESTful接口 | `src/api/` |
| 后台任务 | 异步处理队列 | `src/workers/` |

## ⚙️ 配置

编辑 `config/settings.py` 或使用环境变量：

```bash
# 数据库
DATABASE_URL=postgresql://user:pass@localhost:5432/sentiment

# API密钥
OPENAI_API_KEY=sk-xxx
NEWS_API_KEY=xxx
WEIBO_COOKIE=xxx

# Telegram推送
TELEGRAM_BOT_TOKEN=xxx
TELEGRAM_CHAT_ID=xxx
```
